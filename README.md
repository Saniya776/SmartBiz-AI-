How It Works:
1. Orchestrator (Brain)

Central controller that calls agents, manages state, and compiles outputs.

2. Data Agent

Handles dataset reading, cleaning, and producing structured insights.

3. Reasoning Agent

Simulates business-style logic, decisions, and basic strategy suggestions.

4. Task Agent

Executes supporting workflow tasks.

5. Memory Module

Stores context to maintain decision flow and coherence.
